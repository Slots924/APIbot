# Інструкції для проєкту APIbot

## Опис проєкту
- Скрипт автоматизує взаємодію з Facebook (лайки та коментарі) через локальний AdsPower Browser API, щоб не видаватися ботом.
- Точка входу `src/main.py` запускає профіль AdsPower для вказаного `user_id`, а потім звертається до методів класу `Bot`.
- Клас `Bot` знаходиться у `src/core/bot.py`. Він запускає та зупиняє профіль, підключається до вже відкритого браузера через debugger port, а також делегує дії у модулі `src/core/actions/`.
- У директорії `src/core/actions/` зберігається логіка окремих дій (`like_post.py`, `comment_post.py`). Кожен action містить послідовність Selenium-кроків, що повинні логуатися українською.
- Папка `logs/` зарезервована під майбутній логер (поки що не використовується). Файл `advise.md` містить типові проблеми та поради — **обов’язково** перечитувати перед розробкою нових дій.
- Директорія `testing_fille/` використовується лише для локальних експериментів і не впливає на робочу логіку.

## Вимоги до коду
1. Я новачок, тому кожна функція та важливі блоки коду повинні мати **детальні коментарі українською мовою**, які пояснюють, що і чому робить код.
2. Логи всередині `actions` мають бути українською, на кожен значущий крок (наприклад, «ставлю лайк», «лайк поставлено»).
3. Перед написанням нового `action` обов’язково перегляньте `advise.md`, щоб уникнути типових помилок.
4. Дотримуйтеся існуючої структури: у `bot.py` лише виклики дій, сама логіка знаходиться в окремих файлах `src/core/actions/`.
5. Не вносіть змін до файлів у `testing_fille/`, якщо прямо не зазначено у завданні.

Дякую та приємної роботи! Якщо потрібні уточнення — додавайте їх у коментарях до коду (також українською).
